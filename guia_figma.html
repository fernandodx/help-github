Guia de Referência Figma

Desbrave o Figma: Ferramentas, variáveis, componentes e plugins para o seu fluxo de design.

F O que é Figma?

Figma é uma ferramenta de design de interface e prototipagem baseada na web. Sua principal característica é a colaboração em tempo real, permitindo que várias pessoas trabalhem no mesmo arquivo simultaneamente. É amplamente utilizado para criar interfaces de usuário (UI) e experiência do usuário (UX), sistemas de design e protótipos interativos.

Ele unifica o processo de design, desde a ideação até a entrega para desenvolvimento, em um único ambiente, acessível de qualquer lugar com internet.

Ferramentas Principais

O Figma organiza suas ferramentas de forma intuitiva. Entender cada painel é crucial para um bom fluxo de trabalho:

Painel Esquerdo (Layers & Assets)

  • Layers: Organiza todos os elementos na sua tela, mostrando a hierarquia.
  • Assets: Onde ficam seus componentes, estilos e variáveis para reutilização.
  • Pages: Para organizar diferentes telas ou fluxos do seu projeto.

Painel Direito (Design, Prototype & Inspect)

  • Design: Para ajustar propriedades como cor, tamanho, tipografia, efeitos.
  • Prototype: Conecta frames e cria interações para simular a experiência do usuário.
  • Inspect: Gera códigos CSS, iOS ou Android para desenvolvedores.

Barra de Ferramentas Superior

  • Move Tool (V): Para selecionar e mover objetos.
  • Frame Tool (F): Cria artboards (telas) para seus designs.
  • Shape Tools (R, O, L): Para criar retângulos, círculos, linhas, etc.
  • Pen Tool (P): Para criar formas vetoriais complexas.
  • Text Tool (T): Adiciona caixas de texto.
  • Hand Tool (H): Para navegar pela tela (arrastar).
  • Comments: Adiciona comentários para colaboração.

Variáveis: Cores e Tipografia

Variáveis são a base de um Design System e garantem consistência. No Figma, elas são gerenciadas no painel "Local Styles" ou "Local Variables" (variáveis mais avançadas).

1 Variáveis de Cor

  1. Abra o Painel Design: Clique em qualquer lugar vazio na tela.
  2. Crie um Estilo de Cor: No painel direito (Design), na seção "Color Styles", clique no ícone +.
  3. Defina a Cor: Escolha a cor (hex, RGB, HSL).
  4. Nomeie e Organize por Pasta: Use a notação de barra para criar pastas, por exemplo: cores/primaria/azul-500. Isso criará uma estrutura organizada.

2 Variáveis de Tipografia (Text Styles)

  1. Crie um Texto Exemplo: Use a ferramenta de Texto (T) e escreva algo.
  2. Estilize o Texto: No painel Design, defina fonte, tamanho, peso, altura da linha, espaçamento.
  3. Crie um Estilo de Texto: Na seção "Text Styles", clique no ícone +.
  4. Nomeie e Organize por Pasta: Exemplo: tipografia/heading/h1 ou tipografia/body/regular.

Componentes e Variantes

Componentes são elementos reutilizáveis que você pode usar em várias partes do seu design. Variantes permitem criar diferentes estados ou tipos de um componente principal.

1 Criando um Componente

  1. Desenhe o Elemento: Por exemplo, um botão (retângulo com texto).
  2. Selecione o(s) Elemento(s): Selecione todas as camadas que compõem seu botão.
  3. Crie o Componente: Clique com o botão direito e selecione "Create component" (ou use o atalho Ctrl + Alt + K / Cmd + Option + K), ou o ícone de diamante na barra de ferramentas superior.
  4. Onde Encontrar: Seu componente principal aparecerá no painel "Assets" (esq.).

2 Criando Variantes

  1. Selecione o Componente Principal: Clique no componente que você acabou de criar.
  2. Adicione uma Variante: No painel "Design" (direito), na seção "Variants", clique no ícone + ou "Add variant".
  3. Personalize a Nova Variante: Mude a cor, o tamanho, o texto, etc., para criar um novo estado (ex: botão primário, botão secundário, botão desabilitado).
  4. Defina Propriedades: No componente principal, você pode nomear a propriedade (ex: `tipo`, `estado`) e os valores para cada variante (ex: `primario`, `secundario`, `desabilitado`).

Plugins Essenciais

Plugins estendem a funcionalidade do Figma, acelerando seu trabalho. Para instalar, vá em Menu > Plugins > Find more plugins.

  • Styler: Ajuda a extrair e organizar estilos de cores, textos e efeitos de um design existente. Ótimo para refatorar e criar Design Systems.
  • Unsplash: Adiciona imagens de alta qualidade e gratuitas diretamente do Unsplash para seus designs. Perfeito para placeholders realistas.
  • Typography Style Guide: Gera automaticamente um guia visual com todos os seus estilos de texto. Facilita a documentação da tipografia do seu projeto.
  • Material Icons: Fornece acesso rápido e fácil a milhares de ícones do Material Design da Google, diretamente no Figma.
