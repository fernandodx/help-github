Guia de Referência CSS

Aprenda a estilizar suas páginas web, do básico ao layout avançado.

O que é CSS?

CSS significa Cascading Style Sheets (Folhas de Estilo em Cascata). É a linguagem que usamos para estilizar uma página web. Enquanto o HTML é o esqueleto, o CSS é a pele, as roupas e a aparência.

A principal vantagem do CSS é a separação de responsabilidades: o HTML cuida da estrutura do conteúdo, e o CSS cuida da apresentação visual. Isso torna o código mais limpo, fácil de manter e reutilizável.

Anatomia de uma Regra CSS

p { color: #e2e8f0; font-size: 16px; }
  • p: O Seletor (alvo da regra).
  • { ... }: O Bloco de Declaração.
  • color: #e2e8f0;: Uma Declaração, composta por uma Propriedade (`color`) e um Valor (`#e2e8f0`).

Seletores e Especificidade

Seletores são o coração do CSS. Eles definem quais elementos HTML serão estilizados. A Especificidade é a regra que o navegador usa para decidir qual estilo aplicar quando múltiplos seletores miram no mesmo elemento.

Seletores Básicos

  • Tipo: `p`, `h1`, `div` (seleciona pela tag)
  • Classe: `.minha-classe` (seleciona pelo atributo `class`)
  • ID: `#meu-id` (seleciona pelo atributo `id`, deve ser único)

Especificidade (Hierarquia)

Regras mais específicas sobrescrevem as menos específicas.

  1. Estilos `inline` (atributo `style`) - Mais forte
  2. ID (`#id`)
  3. Classe (`.classe`), Atributo (`[type="..."]`), Pseudo-classe (`:hover`)
  4. Tipo (`div`), Pseudo-elemento (`::before`) - Mais fraco
/* Classe é mais específica que tipo */ .titulo { color: blue; } /* Vence */ h1 { color: red; } /* Perde */ /* ID é mais específico que classe */ #titulo-principal { color: green; } /* Vence */ .titulo { color: blue; } /* Perde */

O Box Model

Todo elemento em uma página web é uma caixa retangular. O Box Model descreve como essa caixa é composta:

  • Content: O conteúdo real (texto, imagem).
  • Padding: Espaçamento interno, entre o conteúdo e a borda.
  • Border: A borda que envolve o conteúdo e o padding.
  • Margin: Espaçamento externo, entre a borda e os outros elementos.
.caixa { width: 200px; padding: 20px; /* 20px em todos os lados */ border: 2px solid #38bdf8; margin: 30px; /* 30px em todos os lados */ box-sizing: border-box; /* DICA: Isso faz o padding e a borda ficarem DENTRO do width! */ }

Layout com Flexbox

Flexbox é um modelo de layout unidimensional, projetado para alinhar e distribuir espaço entre itens em um container, mesmo quando o tamanho deles é desconhecido. É perfeito para componentes e layouts em uma única direção (linha ou coluna).

.container-flex { display: flex; justify-content: space-between; /* Alinha no eixo principal (horizontal) */ align-items: center; /* Alinha no eixo cruzado (vertical) */ }

Propriedades do Container (`pai`)

  • `display: flex;`
  • `flex-direction: row | column;`
  • `justify-content:` (alinhamento horizontal)
  • `align-items:` (alinhamento vertical)
  • `flex-wrap: wrap;`

Propriedades dos Itens (`filhos`)

  • `flex-grow: 1;` (para ocupar espaço)
  • `flex-shrink: 0;` (para não encolher)
  • `flex-basis: 200px;` (tamanho inicial)
  • `order: -1;` (muda a ordem visual)

Layout com Grid

Grid é um modelo de layout bidimensional (linhas e colunas), tornando-o ideal para layouts complexos de página inteira. Você define um "grid" e posiciona os elementos filhos dentro dele.

.container-grid { display: grid; grid-template-columns: 1fr 1fr 2fr; /* Três colunas, a última com o dobro do tamanho */ gap: 20px; /* Espaço entre as células do grid */ } .item-grid { grid-column: 1 / 3; /* Ocupa da coluna 1 até a 3 */ grid-row: 2; /* Fica na linha 2 */ }

Principais Conceitos

  • `display: grid;`
  • `grid-template-columns / rows:` Define o número e o tamanho das colunas/linhas.
  • `fr`: Uma unidade de fração que distribui o espaço disponível.
  • `gap`: O espaçamento entre as linhas e colunas.
  • `grid-column / row:` Especifica em quais linhas/colunas um item deve começar e terminar.
