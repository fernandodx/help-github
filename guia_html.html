Guia de Referência HTML

Um guia visual com a teoria e as principais tags para estruturar suas páginas web.

O que é HTML?

HTML significa HyperText Markup Language (Linguagem de Marcação de Hipertexto). Não é uma linguagem de programação, mas sim uma linguagem de marcação. Sua única função é descrever a estrutura e o conteúdo de uma página web, como um esqueleto.

O navegador (Chrome, Firefox, etc.) interpreta o arquivo HTML, entende as "tags" (etiquetas) e desenha a página na tela para o usuário. Ele constrói uma árvore de elementos chamada DOM (Document Object Model), que representa a estrutura do documento.

Anatomia de um Elemento HTML

Um elemento é formado pela tag de abertura, o conteúdo e a tag de fechamento.

<p class="exemplo">Este é o conteúdo.</p>
  • <p>: Tag de abertura.
  • class="exemplo": Um "atributo", que fornece informações extras sobre o elemento.
  • Este é o conteúdo.: O conteúdo do elemento.
  • </p>: Tag de fechamento.

Estrutura Básica de uma Página

Toda página HTML segue uma estrutura fundamental que informa ao navegador como o documento deve ser interpretado.

<!DOCTYPE html>
<html lang="pt-BR">
<head>
    <meta charset="UTF-8">
    <title>Título da Página</title>
</head>
<body>
    <!-- O conteúdo visível da sua página vai aqui -->
    <h1>Meu Primeiro Título</h1>
    <p>Meu primeiro parágrafo.</p>
</body>
</html>
  • <!DOCTYPE html>: Define que o documento é um HTML5.
  • <html>: O elemento raiz. O atributo `lang` define o idioma da página.
  • <head>: Contém metadados (informações para o navegador), como o título, codificação de caracteres, etc.
  • <body>: Contém todo o conteúdo visível da página.

Principais Tags e Conceitos

Títulos e Parágrafos

Para organizar o texto hierarquicamente.

<h1> a <h6>: Títulos (headings). `h1` é o mais importante.

<p>: Parágrafo.

<h1>Título Principal</h1> <p>Este é um parágrafo.</p>

Links e Imagens

Para conectar páginas e exibir mídias.

<a>: Link (âncora). `href` é o destino.

<img>: Imagem. `src` é o caminho e `alt` é o texto alternativo.

<a href="#">Clique aqui</a> <img src="img.png" alt="logo">

Listas

Para agrupar itens.

<ul>: Lista não ordenada.

<ol>: Lista ordenada.

<li>: Item de lista.

<ul> <li>Item 1</li> </ul>

Formatação de Texto

Para dar ênfase e estilo ao texto.

<strong>, <b>: Negrito. `strong` indica importância.

<em>, <i>: Itálico. `em` indica ênfase.

<u>: Sublinhado (use com cuidado).

<p>Texto com <strong>importância</strong> e <em>ênfase</em>.</p>

Containers: `div` e `span`

Tags genéricas e essenciais usadas para agrupar elementos para estilização ou manipulação.

<div>: Um container em nível de bloco. Ele ocupa toda a largura disponível e força uma quebra de linha. Ideal para agrupar grandes seções de conteúdo.

<span>: Um container em nível de linha (inline). Ele ocupa apenas o espaço necessário e não cria quebras de linha. Perfeito para estilizar uma parte de um texto.

<div style="background-color: #2d3748; padding: 1rem;"> <p>Este parágrafo está dentro de uma div. Ele contém um <span style="color: #f97316;">span colorido</span> no meio.</p> </div>

Atributos Essenciais

Atributos fornecem informações adicionais sobre os elementos e são sempre especificados na tag de abertura.

  • id: Define um identificador único para um elemento. Não pode haver dois elementos com o mesmo ID em uma página. Usado para âncoras e para selecionar um elemento específico com JavaScript.
  • class: Associa um elemento a uma ou mais classes. Vários elementos podem ter a mesma classe. É a forma mais comum de aplicar estilos CSS a múltiplos elementos.
  • style: Permite aplicar estilos CSS diretamente no elemento (CSS "inline"). Deve ser usado com moderação.
<div id="cabecalho-principal" class="container"> <p class="texto-boas-vindas">Bem-vindo!</p> </div>
