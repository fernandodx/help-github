Guia de Referência JavaScript

A linguagem que dá vida à web: do DOM ao Assincronismo.

JS O que é JavaScript?

JavaScript (JS) é uma linguagem de programação de alto nível, interpretada, que roda no navegador do cliente. Enquanto o HTML estrutura o conteúdo e o CSS estiliza, o JavaScript adiciona interatividade e dinamismo às páginas web. Ele permite criar elementos que se atualizam dinamicamente, animações, mapas interativos, e muito mais.

Hoje, com ambientes como o Node.js, o JavaScript também é amplamente utilizado no back-end, tornando-se uma das linguagens mais versáteis do mercado.

Declaração de Variáveis

Variáveis são "contêineres" para armazenar valores. Em JS moderno, temos três formas principais de declará-las:

let

Permite declarar variáveis que podem ter seu valor reatribuído. Possui escopo de bloco (só existe dentro do bloco {} onde foi criada).

const

Declara uma "constante", ou seja, uma variável cujo valor não pode ser reatribuído após sua inicialização. Também possui escopo de bloco. É a forma preferencial de declarar variáveis para evitar reatribuições acidentais.

var

A forma mais antiga de declaração. Possui escopo de função (não de bloco), o que pode levar a comportamentos inesperados. Seu uso é desencorajado em código moderno.

let nome = "Alice"; nome = "Bob"; // Válido const idade = 30; // idade = 31; // Inválido, causaria um erro if (true) { let escopoBloco = "Visível aqui dentro"; const tambemBloco = "Também"; var escopoFuncao = "Vaza do bloco"; } // console.log(escopoBloco); // Erro console.log(escopoFuncao); // "Vaza do bloco" - Funciona!

Funções

Funções são blocos de código reutilizáveis que executam uma tarefa específica. Elas são fundamentais em JavaScript.

Formas de Declarar:

// 1. Declaração de Função (Function Declaration) function soma(a, b) { return a + b; } // 2. Expressão de Função (Function Expression) const multiplica = function(a, b) { return a * b; }; // 3. Arrow Function (Sintaxe moderna e concisa) const subtrai = (a, b) => a - b; console.log(soma(5, 3)); // Saída: 8 console.log(multiplica(5, 3)); // Saída: 15 console.log(subtrai(5, 3)); // Saída: 2

Classes

Classes são um modelo (template) para criar objetos. Elas encapsulam dados e funções que manipulam esses dados. É a base da Programação Orientada a Objetos (POO) em JS.

class Pessoa { // O construtor é um método especial para criar e inicializar um objeto constructor(nome, idade) { this.nome = nome; this.idade = idade; } // Um método da classe apresentar() { console.log(`Olá, meu nome é ${this.nome} e eu tenho ${this.idade} anos.`); } } // Criando uma instância (objeto) da classe Pessoa const pessoa1 = new Pessoa("Carlos", 25); pessoa1.apresentar(); // Saída: Olá, meu nome é Carlos e eu tenho 25 anos.

Métodos Assíncronos

JavaScript é, por natureza, "single-threaded" (executa uma coisa de cada vez). Operações que demoram (ex: buscar dados de um servidor) poderiam travar a página. O assincronismo permite que essas operações ocorram em segundo plano, sem bloquear a interface.

Promises e Async/Await

A forma moderna de lidar com assincronismo é usando Promises, que representam a eventual conclusão (ou falha) de uma operação. A sintaxe async/await foi criada sobre as Promises para tornar o código assíncrono mais legível.

// Função que busca dados de uma API usando async/await const buscarDadosUsuario = async (userId) => { try { // 'await' pausa a execução da função até a Promise ser resolvida const resposta = await fetch(`https://api.example.com/users/${userId}`); if (!resposta.ok) { throw new Error('Erro ao buscar dados'); } const dados = await resposta.json(); // Pausa de novo para processar o JSON console.log(dados); } catch (erro) { console.error("Falha na requisição:", erro); } }; buscarDadosUsuario(1); // Chama a função assíncrona

Manipulação do DOM

O DOM (Document Object Model) é a representação em árvore da estrutura de um documento HTML. O JavaScript pode acessar e modificar essa árvore para alterar o conteúdo, a estrutura e o estilo de uma página.

Principais Métodos:

  • document.getElementById('id'): Seleciona um elemento pelo seu ID.
  • document.querySelector('seletor'): Seleciona o primeiro elemento que corresponde a um seletor CSS.
  • document.querySelectorAll('seletor'): Seleciona todos os elementos que correspondem a um seletor CSS.
  • elemento.addEventListener('evento', funcao): Adiciona um "ouvinte" que executa uma função quando um evento (ex: 'click') ocorre.
  • elemento.textContent / elemento.innerHTML: Altera o conteúdo de texto/HTML de um elemento.
  • elemento.classList.add('classe') / .remove() / .toggle(): Manipula as classes CSS de um elemento.

Exemplo Prático:

Imagine o seguinte HTML:

<h1 id="titulo">Olá Mundo!</h1> <button id="meuBotao">Clique Aqui</button>

E este JavaScript:

// 1. Seleciona os elementos do DOM const titulo = document.getElementById('titulo'); const botao = document.getElementById('meuBotao'); // 2. Adiciona um ouvinte de evento ao botão botao.addEventListener('click', () => { // 3. Ação que acontece quando o botão é clicado titulo.textContent = 'Você clicou no botão!'; titulo.style.color = '#facc15'; // Altera o estilo diretamente });

Ao carregar a página e clicar no botão, o texto do `h1` será alterado e sua cor mudará para amarelo, demonstrando a interação entre JS e HTML.
